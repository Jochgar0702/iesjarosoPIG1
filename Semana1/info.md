Proyecto Intermodular
Explicación del proyecto:
Proyecto de riego automatizado para dos comunidades de regantes
El sistema se basa en una balsa (depósito de agua) que almacena el volumen necesario para riego. Desde esta balsa, una bomba impulsa el agua hacia la red de distribución principal.
Antes de la distribución, el agua pasa por:
Un caudalímetro, que mide cuánta agua se está enviando, y un sensor de presión, que permite verificar que el sistema trabaja dentro de valores seguros.
A partir de ahí, la tubería se divide en dos ramales que alimentan a dos comunidades de regantes:
EV1 → Comunidad 1
EV2 → Comunidad 2
Cada ramal tiene:
Una electroválvula (EV1 y EV2) que abre o cierra el paso del agua,y un sensor de humedad del suelo (SH1 y SH2) que mide cuánta agua necesita cada zona.
